# Agent instructions

## Worktrees
For larger features, a dedicated worktree can make sense, but ask first. Usually it is fine to work directly on main. The worktrees should be created in the root project directory, under .worktrees/

## Git
When Git approval is needed, request persistent approval for git status, git diff, git add, git commit, git fetch, git pull, and git push.
Don’t prompt again for those unless required.

## Other commands
Request persistent approval for these commands:
bash, test, awk, sed, npm, bun
